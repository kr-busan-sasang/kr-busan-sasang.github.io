# 사상구 · 부산광역시

> 사상구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-sasang.github.io/**

사상구(沙上區)는 낙동강 하류 동안에 형성된 넓은 모래벌 위에 자리 잡아 '모래 위'라는 뜻의 지명을 얻은 부산의 대표적 산업·물류 거점입니다. 감전동·학장동·괘법동 일대에 부산 최대 규모의 사상공업지역이 펼쳐져 있고, 부산산업용품유통단지와 서부시외버스터미널이 서부산의 관문 역할을 맡고 있습니다. 도시철도 2호선과 부산김해경전철이 교차하는 사상역을 중심으로 사람과 물자가 모여드는 교통 결절지이기도 합니다. 한편으로는 낙동강변 삼락생태공원과 승학산 억새밭이 공장지대 옆에서 도시민에게 넉넉한 쉼터를 내어 주는 이중의 매력을 지닌 고장입니다. 옛 사상나루와 강선대 당산 신앙이 남아 강과 함께 살아온 주민들의 삶을 오늘까지 전하고 있습니다.
